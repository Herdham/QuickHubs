    HTML Paragraph always start on a new line and it because they are block element i.e block of text

    You can use H1 to H6 to provide the main title of the web page or document


    And Always remember this : HTML stand for -> Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.


    -> HTML Style

    The HTML (p) tags defines an paragraphs

    And remember a paragraph always start on new line is a block element

    HTML Paragraph

    Some paragraph example

    Example

    p This is a paragraph/p
    p This is another paragraph/p

    If you add some extra space and lines the browser will automatically remove all the space when the browser display

    Example

    p This is a paragraph/p
    p 
    This      is another paragraph      containing some white space
    /p

    Remember paragraph is a block element if you write more word inside paragraph tags they will all display on a single line

    Example

    p Twinkle Twinke little start
    How i wonder what you are.
    Up Above the word so high.
    Likea diamond in the sky.
    /p

    -> And to solve this problem use the (pre) element

    -> The pre element allow both white space and line breaks:

    Example

    pre Twinkle Twinke little start
    How i wonder what you are.
    Up Above the word so high.
    Likea diamond in the sky.
