    HTML Introduction

    HTML stand for -> Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.


    And Also: use to structure a web pages, it is a Standard Markup language for web pages. HTML is used to create content and structure of any web page. If you think of the human body as a web page then HTML is the skeleton of the body. It is the building block of web pages. HTML was created by Berners-Lee in late 1991 but "HTML 2.0" was the first standard HTML specification which was published in 1995.

    HTML is the standard markup language for Web pages.

    With HTML you can create your own Website.

    What is HTML


    -> HTML is use to structure a web page

    -> HTML Consist series of tags and different layout

    -> HTML Consist many series of elements like "heading", "paragraphs", "links", and more.


    A Simple HTML Document

    Example

    !DOCTYPE html

    html

    head

    title Page title/title

    /head

    body

    h1 This is heading tag/h1
    p This is paragraph./p

    /body

    /html

    Example Explained

    • The (!DOCTYPE HTML) declaration defines that this document is HTML5 Documents
    • The (Html)element is the root element of an HTML page
    • The (head) element contains all meta information, link, script tag and more.
    • The (title) specifies the title of the HTML Page (which show's in the browser's title bar or page's bar).
    • The (body) element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
    • The (h1) element defines the large heading tags
    • The (p) element defines a paragraphs
