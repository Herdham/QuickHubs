    HTML attributes provide additional information about HTML elements.


    And Always remember this : HTML stand for -> Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.


    -> HTML Attributes

    • All HTML elements can have attributes
    • Attributes provide additional information about elements
    • Attributes are always specified in the start tag
    • Attributes usually come in name/value pairs like: name="value"

    The (a) tags in example below defines a hyperlink. The href attribute specifies the URL of the page the link goes to:
