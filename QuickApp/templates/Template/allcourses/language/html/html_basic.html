    In this chapter will are going to teach you some basic HTML Example

    Don't get confuse if we use some tags it's not that complex but un-familiar


    And Always remember this : HTML stand for -> Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.


    HTML Document

    -> It is a best and good pratice to start your html document with document declaration which (!DOCTYPE html)

    -> The HTML Document itself begins with (html)

    -> And the visible part of the document must begin with (body)

    Example

    !DOCTYPE html

    html

    head

    title Page title/title

    /head

    body

    h1 This is heading tag/h1
    p This is paragraph./p

    /body

    /html

    The (!DOCTYPE Html) Declaration tags

    It's define the type of the page and also makes the web page display correctly.

    The (!DOCTYPE Html) tags must be at the top of the file once the file is created.

    The declaration type for (!DOCTYPE Html) is:


    !DOCTYPE html


    The Headings tags

    HTML Heading tags define with the (h1) to (/h6) tags

    The (h1) to (h1) are the most important tags in HTML

    h1 This is heading 1/h1
    h2 This is heading 2/h2
    h3 This is heading 3/h3

    The Paragraph tags

    HTML Paragraph define with tags

    p This is a paragraph/p
